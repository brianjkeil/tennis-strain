Tennis Strain Monitor

Privacy Policy

Effective date: April 19, 2026  ·  Personal use application
This is a personal productivity tool built for one user. It does not collect, share, sell, or transmit any data to any third party.

What this app does

Tennis Strain Monitor is a personal health tracking application that connects to the WHOOP API to retrieve workout strain data. It is used exclusively by its owner, Brian Keil, to monitor and analyze tennis training sessions.

Data collection

This application accesses the following data from your WHOOP account via the official WHOOP API:

· Workout activity data (activity name, strain score, duration, start time)

No other WHOOP data is accessed or stored.

Data storage

All data is stored exclusively in your browser's local storage on your own device. No data is transmitted to any server, database, or third-party service. No data leaves your device.

Data sharing

This application does not share any data with any person, company, or service. There are no analytics, no advertising, no tracking pixels, and no external data connections of any kind beyond the WHOOP API used to retrieve your own personal data.

WHOOP API access

This application uses the WHOOP Developer API under WHOOP's standard OAuth 2.0 authorization framework. Access tokens are stored locally on your device and are used only to fetch your personal workout data. You can revoke access at any time through your WHOOP account settings.

Data retention

Data persists in your browser's local storage until you manually clear it or uninstall the application. No backups or copies are made anywhere else.

Your rights

As the sole user of this application, you have complete control over all data. You can delete all stored data at any time by clearing your browser's local storage or using the clear data option within the app.

Contact

This is a personal tool. For questions about WHOOP data specifically, refer to WHOOP's own privacy policy at whoop.com.
