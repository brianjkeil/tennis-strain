Setup guide

Complete these steps once. After that, sync takes one tap.

Step 1Go to developer.whoop.com and create a developer account using your existing WHOOP email.
Step 2Click Create App. Name it anything — "Tennis Monitor" works.
Step 3Set the Redirect URI to exactly the URL shown in the App Credentials panel on the left. Copy it using the Copy button.
Step 4Set the Privacy Policy URL to your GitHub Pages URL + /privacy-policy.html — e.g. https://yourusername.github.io/tennis-strain/privacy-policy.html
Step 5Copy your Client ID and Client Secret from the WHOOP portal and paste them into the App Credentials panel.
Step 6Click Save credentials, then Connect WHOOP. Authorize in the popup. Done.
Step 7From now on, just click Sync now whenever you want to pull in new sessions. Takes a few seconds.
Required WHOOP API scopes
When authorizing, make sure read:workout is checked. That's the only scope this app needs.
